Starting a react project example. Women-in-Stem-Website
Our Github repo: https://github.com/avryjacksoncom/women-in-stem-club-website
 Guide to run project: 

 Download github desktop
https://desktop.github.com/download/

 download git 
https://git-scm.com/downloads

 download npm 
https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

 Download Node.js 
https://nodejs.org/en/download

 Clone the repository: 
https://github.com/avryjacksoncom/csc481project.git

In Desktop github open with your code editor.

USE COMMAND PROMPT ON WINDOWS
it defaults to powershell, commands do not work with power-shell so switch. 
 
Open up the terminal in vscode.
Then type this command in terminal:  cd fullstack-front
This just travels into our directory folder where our project is.
Then type:  npm install 
This installs the dependencies we are using.
Then:  npm start
This is to start to run our project.
I think to end the local host connection you just do ctrl + c or something.
 You only need to do npm install one time. You don't have to run that multiple times.
